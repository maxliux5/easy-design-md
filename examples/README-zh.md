# EASY DESIGN.MD

告诉 AI 你想模仿哪个品牌，它会自动为你下载对应的 **DESIGN.md** 文件。支持 59+ 个设计系统。

---

## 我们的原则

### 先展示，再下载

**永远先展示 Demo，让用户在下载前做出选择。**

当用户说"我想做一个页面"时，不要盲目下载。正确的流程是：

1. 展示交互式 Demo
2. 让用户**看到推荐品牌的配色和预览效果**
3. 等待用户选择
4. 然后再下载

这样确保用户基于视觉结果（而不是品牌名称）做出明智的决定。

### 交互式 Demo

在线体验：[color-recommender.html](examples/color-recommender.html)

选择页面类型 → 查看推荐品牌的调色板 → 实时预览效果 → 有信心地选择。

## 功能特点

| 功能 | 说明 |
|-----|------|
| **Demo 优先** | 下载前先展示视觉效果 |
| **智能推荐** | 告诉 AI 页面类型，AI 匹配最合适的品牌 |
| **59+ 设计系统** | Stripe、Linear、Airbnb、Apple、Figma、Vercel、GitHub 等 |
| **一键下载** | 瞬间获取任意品牌的设计系统 |

## 安装

```bash
npx skills add maxliux5/easy-design-md
```

然后告诉 AI：**"我想做一个 SaaS 官网"**

---

## 使用方法

安装后，在 Claude Code 中使用：

```
/easy-design-md
```

工作流程：
1. AI 展示 Demo（见上方视频）
2. 你选择一个设计系统
3. AI 下载 DESIGN.md 文件
4. AI 基于该设计语言为你构建页面
